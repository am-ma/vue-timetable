vue-timetable
=========

- タイムテーブルの進捗を管理するアレ
- 5分おきに終わったかどうか？ をみに行きます
- 勉強会の次の発表なにかな？ などを見るのに欲しかったので作りました

usage
------
テキストエリアにテーブルの文字列をコピペ（列はTAB区切り、行は改行区切り）

テーブルのサンプル
------
時間 | 発表者 | 説明
---|-----|---
19:00 - 19:15 | 田中 | Vue.jsについてのなにか
19:15 - 19:30 | 鈴木 | 非同期処理についてのなにか
19:30 - 19:45 | 佐藤 | Vue.jsで○○してみた
20:00 - 21:00 | 懇親会
